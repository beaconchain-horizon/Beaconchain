## 🧠 What is Beaconchain Horizon?

**Beaconchain** is an **open-source, offline-first blockchain monitoring platform**.  
**Horizon** is the product suite built on this platform, featuring:

- **Go** – Backend API
- **SQLite / PostgreSQL** – Database
- **Merkle Trees** – License generation and verification
- **ECDSA** – Digital signatures for license integrity
- **Gin** – Web framework
