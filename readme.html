📖 The Story

Beaconchain is not just a monitoring dashboard – it is the birthplace of the Horizon ecosystem. Founded by Mahdi Amolimoghaddam in 2017, this project started as a personal journey to bring transparency, security, and decentralization to Ethereum staking. Today, it stands as the world's #3 open-source blockchain monitoring platform with a perfect 10/10 security score (May 2026).

This repository contains the core backend engine, Merkle tree library, license management, and API services that power the entire Horizon ecosystem – including the Beaconchain.us dashboard, the Horizon Store, and the upcoming Horizon Hub.

🏛️ Architecture Overview

This repository contains the core components of the Beaconchain backend:

horizon-core-engine/
├── cmd/
│   ├── core-engine/     # Main API server
│   ├── license-gen/     # Manual license issuance CLI
│   └── license-check/   # Offline verification CLI
├── internal/
│   ├── license/         # License logic + Merkle integration
│   ├── product/         # Product catalog
│   ├── ipmanager/       # IP pool management
│   └── storage/         # SQLite layer
├── pkg/
│   ├── merkle/          # Merkle tree library
│   └── crypto/          # ECDSA signing library
├── data/
│   ├── licenses.db      # SQLite database
│   └── ip_pool.json     # Available IPs
├── .env.example
├── go.mod
└── README.md

Key Technologies

  • Language: Go 1.21+
  • Storage: SQLite (offline) + Redis (optional)
  • Cryptography: Merkle Tree, ECDSA (P‑256), SHA‑256
  • License Model: PrepaidPackage – volume‑based, one‑time use leaves

API Endpoints

  • /init-keys – Initialize validator keys
  • /validators – List all validators
  • /license/generate – Generate a new license
  • /licenses – List all licenses
  • /verify – Verify license authenticity
  • /health – Health check

🚀 Quick Start

Prerequisites

  • Go 1.21+
  • SQLite (optional, embedded)
  • Git

Clone & Run

# Clone the repository
git clone https://github.com/beaconchain-horizon/beaconchain.git
cd beaconchain

# Copy environment file
cp .env.example .env

# Download dependencies
go mod download

# Run the core engine
go run cmd/core-engine/main.go

The API server will start on http://localhost:8080 by default.

Environment Variables

Create a .env file with the following:

DATABASE_PATH=./data/licenses.db
IP_POOL_PATH=./data/ip_pool.json
PORT=8080
